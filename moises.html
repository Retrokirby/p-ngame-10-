mi parte

En el proyecto de cuarto periodo, trabajé en la estructura y funcionalidad de una página web para gestionar la compra de apartamentos al contado. Me encargué de implementar el HTML para crear el formulario de registro, en el cual se recogen datos como nombre, apellido, edad, teléfono, salario y el tipo de apartamento elegido. Además, desarrolló la función en JavaScript que calcula el precio final del apartamento según el tipo seleccionado, aplicando un descuento específico para cada categoría. Este cálculo se realiza al enviar el formulario, garantizando que el cliente reciba la información del costo total con el descuento.


lo que aprendi


Este año aprendí a usar herramientas y lenguajes esenciales para el desarrollo de software. Utilizamos SQL Server y NetBeans para crear bases de datos funcionales , lo cual me permitió comprender mejor cómo estructurar y gestionar la información de forma efectiva. También aprendí HTML y JavaScript, que me ayudaron a construir interfaces web y agregarles funcionalidad interactiva.
